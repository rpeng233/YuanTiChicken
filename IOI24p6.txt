First reconstruct the maximal connected blocks of equal hidden value. Maintain a DSU of already known blocks. Process vertices in order; set all unprocessed vertices to a dummy value and leave processed ones unchanged. Compare the returned component count with the count predicted by the current DSU model. If they differ, the new vertex belongs to some existing block. Binary-search over existing block representatives by leaving only half of them unchanged and setting the rest to the dummy value; the half that still causes the discrepancy contains a block to merge. Union it with the new vertex and repeat until the prediction matches.

Then collapse each discovered block into one node. If there is only one node, try each possible value on one representative; the value that leaves the whole structure connected is the answer.

Otherwise, build a spanning tree of the collapsed graph and split its nodes by depth parity. For each value `x` and each parity side, find all still-unknown nodes on the opposite side with value `x`: set the current side and all already-solved nodes to `x`, leave the target side unchanged, and compare the real result with the simulated one. A mismatch means at least one target node has value `x`. Binary-search among target nodes in the same way, leaving half unchanged and setting the rest to `x`, until one such node is found; assign it `x` and repeat. After both parity sides and all values are processed, expand the collapsed-node values back to the original vertices. The number of experiments is `O(N log N)` and fits the limit for `N ≤ 250`.
